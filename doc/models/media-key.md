
# Media Key

Identifying key information for a media asset.

## Structure

`MediaKey`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `id` | `String` | Required | Unique ID for a known media asset. |

## Example (as JSON)

```json
{
  "id": "asset/399eb0ed4156fd3f/original/quwxghjbnop30rmzl0tx"
}
```
