
# Error Detail

Detailed information about an error.

## Structure

`ErrorDetail`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `errorcode` | `String` | Required | Specifies an error code or well-qualified error path. |

## Example (as JSON)

```json
{
  "errorcode": "\"oauth.v2.InvalidApiKey\""
}
```
